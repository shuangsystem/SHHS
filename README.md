

# Shhs 项目

## 简介

Shhs 是一个基于自定义函数式编程语言的项目，提供了一套核心的标准库定义，仿照 Haskell 的经典数据结构与类型类实现，为数学逻辑、集合运算、数值计算等常用场景提供基础工具。该项目采用纯文本 `.shhs` 文件作为源码，统一存放于仓库根目录以及 `Data/` 子目录中，便于模块化管理与扩展。

## 项目规模与结构

```
.
├── Data/
│   ├── Logic.shhs     # 逻辑运算相关定义
│   ├── M.shhs        # 单子（Monad）相关的类型与函数
│   ├── Num.shhs      # 数值类型类与基本算术运算
│   ├── P.shhs        # 多态与参数化类型的支持
│   └── Set.shhs      # 集合数据类型及集合操作函数
├── Prelude.shhs      # 基础预lude定义，包含常用的通用函数
├── haskelldef.shhs   # 与 Haskell 语法兼容的定义层
├── all.txt           # 模块导入汇总表
└── LICENSE          # 开源许可证文件
```

其中：

- **Data/** 目录下的各个模块对应不同的功能域，分别负责逻辑、集合、数值等基础抽象的实现；
- **Prelude.shhs** 是整个语言环境的入口文件，提供所有其它模块所依赖的核心函数与类型；
- **haskelldef.shhs** 用于桥接本语言与标准 Haskell 的类型系统，使用户能够复用已有的 Haskell 代码；
- **LICENSE** 明确了项目的开源许可条款。

## 主要特性

- **纯函数式**：全部实现不包含副作用，遵循函数式编程的不可变性原则；
- **类型类（Typeclass）**：通过类型类实现多态，包括 `Eq`、`Ord`、`Functor`、`Monad` 等常用类；
- **集合运算**：提供基本的集合数据类型及其交、并、补等运算；
- **数值支持**：实现了整数、有理数等数值类型的常见算术操作；
- **模块化设计**：各功能模块相互独立，按需导入即可使用；
- **与 Haskell 兼容**：通过 `haskelldef.shhs` 层可以实现与 Haskell 标准库的互操作。

## 快速开始

### 环境要求

- 任何支持 UTF‑8 编码的文本编辑器；
- 解释器或编译器（如有官方提供的 `shhs` 运行环境）；
- Git（用于克隆仓库）。

### 安装步骤

```bash
# 1. 克隆仓库
git clone https://gitee.com/shuangsystem/shhs.git
cd shhs

# 2. 查看项目结构（可选）
ls -R
```

如果项目提供了可执行脚本，请按照项目根目录的 `README` 或 `INSTALL` 文件中的说明进行编译或解释运行。

### 简单使用示例

下面给出加载 ` Prelude ` 并进行基本数值计算的最简示例（具体语法请参阅对应模块的内部注释）：

```shhs
-- 引入 Prelude 中的数值类型类
import Prelude

-- 使用 Num.shhs 中定义的加法 (+) 与乘法 (*)
let result = 3 + 4 * 2
in  show result
```

## 模块说明

| 文件 | 功能描述 |
|------|----------|
| **Data/Logic.shhs** | 提供布尔值 (`Bool`) 以及常用逻辑运算符 (`&&`、`\|\|`、`not` 等) 的实现。 |
| **Data/M.shhs** | 定义单子（`Monad`）的类型类及相关操作符，用于链式计算和状态传递。 |
| **Data/Num.shhs** | 实现整数与有理数的算术运算、比较运算符以及数值类型类实例。 |
| **Data/P.shhs** | 多态函数的参数化支持，帮助构建通用的算法模板。 |
| **Data/Set.shhs** | 集合数据结构的创建、交集、并集、差集等基本集合操作。 |
| **Prelude.shhs** | 所有其他模块的公共依赖，包含最基本的函数、数据类型和类型类。 |
| **haskelldef.shhs** | 为兼容 Haskell 的类型推导提供别名与桥接定义，降低迁移成本。 |

> **提示**：每个 `.shhs` 文件内部都有对应的注释与示例，建议在实际使用前先阅读相应文件的文档字符串。
