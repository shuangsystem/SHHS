# SHHS (SHuang HaSkell)

**SHHS 是一个使用中文词汇和部分英文关键字的人造语言, 具有严谨的语法规则**

SHHS 脱胎于haskell, 设计目的为无歧义, 高效地表达语义

---

## SHHS 做了什么?

SHHS 定义了2种主要类型:

`S a`

**提问/值集合**：一个集合, 作为句子时的功能为提问集合内容


`M`

**陈述/事实累积**：一串陈述, 本质上是'原子陈述'的集合

通过 `which`, `what`, 对自然语言中的名词性从句提供了抽象

---

## 简单的例子

```haskell
-- "我保护你"
保护 你 我

-- "我保护你，弄死欺负你的人"
main :: M
main = overAll do
  p <- what $ 欺负 你
  pure 保护 你 我 <> pure 弄死 p 我
```
